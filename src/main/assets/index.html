Gemini PDA

Japanese Key Re:Map

Description

本アプリは、Planet Computers Android端末の日本語キーボードのマッピングを刻印の通りに入力できるようにするための、アプリです。
刻印通りの入力を目指していますので、: や - の入力などは、刻印の通りに入力できます。

キーマップを変更した場合には、新しいキーマップを認識するため「再起動」を実行してからご利用ください。

Support Hardware

本アプリは、以下の端末で動作確認を行いました。

  • Gemini PDA
  • ASTRO SLIDE 5G Transformer

Uninstall

本アプリをアンインストールすると、キーマップはGemini PDA搭載のキーマッピングに戻ります。
日本語入力のリマップを使用する場合には、本アプリをインストールしたままでご利用ください。

本文章アプリに記載されている会社名、商品名、製品名などは、一般に各社の商標もしくは登録商標です。
